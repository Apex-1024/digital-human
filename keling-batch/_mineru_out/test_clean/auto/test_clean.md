## 问题：变速直线运动中位置函数与速度函数的联系

设某物体作直线运动，速度是 $\nu = \nu ( t )$ ，且 $\nu ( t ) \geq 0$ ，在时间段 $[ T _ { 1 } , T _ { 2 } ]$ 内：

运动路程为： $\int _ { T _ { 1 } } ^ { T _ { 2 } } \nu ( t ) d t$

又设位置函数为s(t)，则路程为： $s ( T _ { 2 } ) - s ( T _ { 1 } )$

$$
\therefore \int _ { T _ { 1 } } ^ { T _ { 2 } } \nu ( t ) d t = s ( T _ { 2 } ) - s ( T _ { 1 } ) .
$$
